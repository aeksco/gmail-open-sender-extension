:envelope: Adds a button to Gmail's message view to open all messages from that sender in search

**Getting Started**

Run the following commands to install dependencies and start developing

```
pnpm install
pnpm run dev
```

**Scripts**

-   `pnpm run dev` - run `webpack` in `watch` mode
-   `pnpm run build` - builds the production-ready unpacked extension
-   `pnpm run lint` - runs EsLint
-   `pnpm run prettify` - runs Prettier

**Notes**

-   Includes ESLint configured to work with TypeScript and Prettier.
